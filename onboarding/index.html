Mission 7

Deployment Instructions

  1. Move brain-forge-export.json to onboarding/context/brain-forge-export.json
  2. The /onboard-apply command should already be at .claude/commands/onboard-apply.md
  3. Open Claude Code in your brain template repo
  4. Run /onboard-apply
  5. Claude reads your data, asks clarifying questions, and populates all brain files
Brain Insight
/onboard-apply is itself a skill -- just like the /roadmap you built in Mission 6. You describe what you want in plain language, Claude handles the formatting.
